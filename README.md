## 📖 Project Overview

EnterpriseRisk-Pro is an enterprise risk profiling and early warning system designed for financial institutions.

The project integrates multi-source public enterprise data, including:

- Business registration information
- Financial indicators
- Judicial records
- Administrative penalties
- Public opinion data

to build enterprise risk profiles and generate intelligent risk scores through machine learning models.

---

## 🎯 Project Objectives

- Build enterprise risk profiles
- Develop enterprise risk scoring models
- Detect potential enterprise risks
- Generate automated risk reports
- Visualize enterprise risk dashboards
