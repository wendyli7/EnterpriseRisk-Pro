# EnterpriseRisk-Pro 项目上下文说明

## 1. 项目基本信息

项目名称：

基于多源公开数据的企业风险画像构建与风险预警模型研究

英文名称：

EnterpriseRisk-Pro

项目定位：

面向金融风控场景，通过采集企业工商、司法、经营、舆情等多源公开数据，构建企业风险画像体系，并建立企业风险评分模型，实现企业风险识别、风险预警和风险分析。

项目目标：

1. 构建企业风险数据仓库
2. 完成企业多维度风险指标体系设计
3. 实现企业风险评分模型
4. 提供风险查询和预警接口
5. 搭建可视化风险画像展示系统

应用场景：

- 金融机构贷前审核
- 企业信用风险评估
- 供应链金融风险管理
- 企业尽调分析
- 风险监测预警

## 2. 项目背景

项目当前主要用于：
